# 🧭 Путь по СНГ — Landing Page турагентства

Одностраничный сайт-визитка туристического агентства, специализирующегося на турах по странам СНГ. Учебный проект по дисциплине «Программирование».

## 🌐 Демо

[Открыть сайт](https://ripeeerr.github.io/html-prac/)

## 📋 О проекте

Коммерческий лендинг туристического агентства **«Путь по СНГ»** — компании, организующей маршруты по восьми странам: России, Казахстану, Кыргызстану, Узбекистану, Армении, Грузии, Беларуси и Азербайджану.

Сайт реализован без сторонних фреймворков и библиотек — только нативный HTML5, CSS3 и JavaScript.

## 🛠️ Технологии

| Технология | Применение |
|---|---|
| HTML5 | Семантическая разметка (header, main, section, article, footer) |
| CSS3 | Кастомные свойства, Grid, Flexbox, анимации, медиазапросы |
| JavaScript (ES6+) | Слайдер, фильтр туров, модальное окно, формы, IntersectionObserver |
| Google Fonts | Cormorant Garamond (заголовки) + Inter (текст) |
| Unsplash | Фотографии туров и слайдера |

## ⚙️ Функциональность

**Слайдер**
- Автоматическая смена каждые 4.5 секунды
- Кнопки «назад» / «вперёд»
- Навигационные точки с кликом

**Фильтр туров**
- Мгновенная фильтрация по стране через `data-country`
- Активное состояние кнопки-фильтра

**Модальное окно бронирования**
- Открывается с кнопки «Забронировать» на любой карточке тура
- Название тура автоматически подставляется в селект
- Закрытие по клику на оверлей или клавише Escape
- Показывает сообщение об успехе и закрывается через 3 секунды

**Формы**
- Форма подписки: показывает подтверждение, сбрасывается через 4 секунды
- Форма бронирования: имитация отправки без реального бэкенда

**Прочее**
- Кнопка «↑» появляется после прокрутки на 400px
- Шапка меняет фон при скролле
- Анимации появления элементов через `IntersectionObserver`
- Бургер-меню на мобильных устройствах

## 🎨 Дизайн

Цветовая схема построена на природной палитре:

```
--clr-dark:   #1A2E1A  /* тёмно-лесной  */
--clr-forest: #2D5A27  /* хвойный зелёный */
--clr-leaf:   #7EB87A  /* мягкий зелёный  */
--clr-cream:  #F7F3EE  /* тёплый молочный */
--clr-amber:  #D4A853  /* янтарный акцент */
```

Адаптивность: корректное отображение на экранах от 320px до 1440px+.

## 🚀 Запуск

Достаточно открыть `index.html` в браузере — сборка не требуется.

```bash
git clone https://github.com/Ripeeerr/html-prac.git
cd html-prac
# открыть index.html в браузере
```

Для живой перезагрузки при разработке удобно использовать расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code.
